Hi 👋 My name is Volodymyr Sakharov
===================================

Java Software Developer | Devops
--------------------------------

I started my journey with embedded systems during school in Ukraine, working with Arduino (C, C++) and Raspberry Pi (C, Python), which sparked my love for C++. After school, I pursued my studies in Poland, focusing on C/C++, Assembler, and C#. It was during this time that I became passionate about Java. Alongside my university courses, I started self-learning various Java frameworks like Spring Boot and Hibernate and gained experience with both SQL and NoSQL databases. Since 2021, I've been professionally developing web applications using Java, Spring Boot, Hibernate, and databases. In 2023, I expanded my skill set by learning Docker, Kubernetes, and Jenkins, and have been actively working in the DevOps sphere, integrating these tools into my development workflow.

*   🌍  I'm based in Zielona Góra, Poland, want to relocate to Poznań
*   ✉️  You can contact me at [savolodya@gmail.com](mailto:savolodya@gmail.com)
*   🧠  I'm learning kubernetes, java, databases
